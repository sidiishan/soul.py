## 📋 What is soul.py?

soul.py helps large language models (LLMs) remember and keep their identity over time. It works with any LLM provider and saves information in markdown format. This lets your AI agents recall past conversations or actions without starting fresh each time. 

This tool is useful if you want a chatbot or AI assistant that feels consistent. It keeps track of what the AI knows and keeps its personality stable. If you use apps based on LLMs, soul.py can keep them smarter and more personal.
